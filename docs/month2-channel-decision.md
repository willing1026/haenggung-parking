### 각 옵션 상세

**카카오 채널**
- 장점: 한국인 단골에게 친숙한 인터페이스, 알림 카톡으로 수신
- 단점: 개설 시 사업자/본인 인증 필요, 모바일 앱 전환 단절, 50명 초과 시 과금
- 활성화 방법: `KakaoSubscribeBanner.tsx`의 `DISABLED = false` + `COPY.url` 교체

**이메일 (Stibee)**
- 장점: 도달 알고리즘 없음, 오픈율/클릭률 측정 가능, 500명 무료
- 단점: 이메일 주소 입력 마찰, 확인 메일 단계 추가 가능
- 활성화 방법: 배너 URL을 Stibee 구독 폼 URL로 교체 + `DISABLED = false`

**Web Push**
- 장점: friction 가장 낮음 (브라우저 허용 1클릭)
- 단점: iOS Safari 미지원 (한국 모바일 사용자 상당수 제외), 사용자 거부율 높음
- 활성화 방법: 별도 Web Push 서비스 연동 필요 (OneSignal 등)

**즐겨찾기 유도**
- 장점: 운영 부담 없음
- 단점: "허락"이 아닌 "의도" — 재방문 보장 안 됨, 측정 불가
