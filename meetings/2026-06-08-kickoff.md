## 의제 1: 한국인 단골 vs 글로벌 관광객 — 우리 부족은 누구인가?

### 각 에이전트 입장 (요약)

**tribe-intel — 한국인 단골 우선**
- SNS 언어 분포: 한국어 약 85~90% / 영어·기타 약 10~15%. "행리단길" 자체가 한국어 신조어 → 외국인에게 발음·의미 불투명. 행궁동 골목상권과 수원화성 관광은 동선이 겹치되 목적이 다르다 (전자: 로컬 카페·맛집 / 후자: 문화유산).
- 내면 서사 적합도: "남들보다 잘 안다"는 비교 준거가 필요 → 한국 단골에게만 작동. 글로벌 관광객의 서사는 "경험해봤다"이지 "잘 안다"가 아님. 행궁동은 그들에게 체크포인트이지 부족 귀속의 장소가 아님.
- 부족 규범("미리 챙기고 알려준다") = 한국 정보 문화(네이버 사전검색·카카오맵·카톡방)와 정렬. 글로벌 관광객의 정보 탐색은 TripAdvisor·Google·Airbnb 중심이라 우리 사이트가 접점이 되기 어렵다. 공유 경로도 본국 SNS·영어권 포럼으로 빠지면 재유입 단절.
- 한 줄 입장: **"글로벌 관광객은 트래픽이지 부족이 아니다."**

**content — 한국인 단골 압승**
- 리마커블 기준 통과는 한국어가 압도적 유리. "카톡방 공유" 매개 자체가 한국인 중심.
- 4가지 콘텐츠 유형(현지인 관점·실시간 인텔·역발상·부족 큐레이션) 모두 한국 단골 톤에서 진정성이 살아남. 영어 관광객 톤은 TripAdvisor 톤과 차별화가 어렵고 AI 생성 시 일반화·진부화 위험 큼.
- 실시간성·역발상은 한국 현지인만이 의미 있는 정보를 줄 수 있다.

**permission-ladder — 한국인 단골 압도적 우위**
- 카카오 채널은 한국인에게 friction 0에 가까움. 글로벌 관광객 대안(이메일 / X / 인스타)은 전환율이 1/5~1/10 수준.
- 재방문 동기: 한국 단골은 "여러 번 가는 곳" → 허락 사다리 모든 단계에서 전환율 우위. 글로벌 관광객은 "한 번 가는 곳" → 옹호자 단계에서 구조적으로 막힘.
- UGC 옹호자 발굴 가능성: 한국 단골은 제보 가능, 글로벌 관광객은 사실상 불가능.
- "사라지면 아쉬워할 사람" 테스트: 단골에게만 통과 가능.

**seo-deploy — 한국어 타겟 우선 권고**
- 키워드 검색량: 한국어 "행궁동 주차"/"행리단길"이 영어 "Haenggungdong parking" 대비 약 10~50배. "행궁동"이라는 키워드 자체가 영어로는 존재감이 거의 없음.
- 경쟁 빈틈: 한국어 검색에서는 네이버 블로그/지도 강력, 그러나 "실시간 주차 잔여석" 정보는 빈틈. 영어 검색은 경쟁자 적지만 수요 자체가 미미.
- 인덱싱: Google 인덱싱은 안정적, 네이버는 별도 등록·블로그 교차 발행이 필요. hreflang/i18n은 Phase 1에서 기술 부채만 큼.
- 사이트 인프라 보고(이미 라이브): /about, /contact, /privacy, sitemap.xml, robots.txt, opengraph-image 모두 완성. AdSense 신청 가능 상태.

**analytics — 수익화·측정 관점 (한국 우선 지지)**
- 애드센스 CPC/RPM: 한국 CPC가 글로벌(미국 등) 대비 낮으나, 트래픽 규모가 더 빨리 차서 절대 수익은 한국 쪽이 빠르게 누적됨.
- 트래픽 100/일 도달 속도: 한국 타겟 1~2개월, 글로벌 타겟 6개월+ 추정.
- 부족 건강도 측정: 카카오 채널 구독자 수치를 GA 이벤트와 결합해 추적 가능. 글로벌은 GA only로 사각지대가 큼.

### 크로스 비판 (요약)

5인 만장일치였기에 형식적 비판보다는 **"한국인 우선 결정이 잘못될 수 있는 경로"** 를 팀장이 직접 점검:

- **반박 가능성 1 — "한국 시장은 너무 작다"**: PV 1만/월 천장에 빨리 부딪힐 수 있음. → 반박: Month 1 목표가 PV 1,000이고, 우리는 부족 깊이가 일차 지표다. 천장은 부족이 깊어진 후에 푼다.
- **반박 가능성 2 — "글로벌이 CPC가 높다"**: 맞다. 그러나 부족 0명에서 글로벌 트래픽 끌어오는 비용이 CPC 차이를 압도. 부족 먼저, 수익화는 따라온다.
- **반박 가능성 3 — "이중 타겟 가능?"**: 분산은 부족을 약화시킨다. "타깃을 모든 방문자로 정의"는 팀장 금기 항목. **거부.**

### 팀장 결정

> **결정: 1차 부족은 "수원·경기 거주 행궁동 단골 한국인"으로 확정한다.**

**근거 (Seth Godin 체크리스트 통과):**
1. **부족에 봉사하는가?** Yes. 카톡방에 공유될 가치 = 한국 단골에게만 작동.
2. **리마커블한가?** Yes. 실시간 인텔·역발상은 한국 단골 톤에서만 진정성이 산다.
3. **퍼널이 작동하는가?** Yes. 카카오 채널 + 재방문 동기 + UGC 옹호자 발굴 — 모든 단계가 한국인에서 닫혀 있다.
4. **딥인가?** Yes. 한국 단골은 비교 대상이 있어 "잘 안다"는 정체성을 살 수 있다.

**문 열어둘 것 (피벗 트리거):**
- Month 3 회고 시점에 "수원화성 영어 관광객 주차" 키워드만 별도로 측정한다.
- 만약 글로벌 트래픽이 의도하지 않았는데도 10%/월 이상 들어오면, Month 4에 별도 i18n 분기를 검토한다.
- 그 전까지는 한국어 단일 타겟. hreflang/번역 작업 일체 금지.

## 의제 2: Week 1 실행 계획 (2026-06-05 ~ 06-11, D+3)

### 인프라 상태 재평가

seo-deploy 보고로 PLAN.md의 Week 1 SEO 셋업이 **이미 라이브** 상태임을 확인:
- /about · /contact · /privacy 콘텐츠 완성
- sitemap.xml 자동생성 작동
- robots.txt 최적화 완료
- OG 이미지 설정 존재

PLAN.md Week 1 체크리스트의 SEO/배포 항목 5개는 **사실상 완료**. 시간이 빡빡한 게 아니라 **다음 단계로 더 빠르게 넘어갈 여유가 생겼다.**

### Week 1 잔여 우선순위 (D+3 ~ D+6, 4일 안에)

**P0 (반드시 6/11까지)**
1. **콘텐츠 씨앗 발행 2~3편** (content)
   - "행궁동 주차 완전정복 — 시간대별 꿀팁" (실시간 인텔 톤)
   - "행리단길 카페 단골이 고른 TOP 5" (현지인 관점)
   - "이번 주 행궁동" 첫 편 (실시간 인텔)
   - 모든 글 하단 제보 유도 문구
2. **카카오 채널 개설 + 사이트 배너** (permission-ladder)
   - 카카오 채널 개설 → 채널 URL 확보
   - 사이트 우측 하단 fixed 배너 or 콘텐츠 하단 CTA 카드 1종
   - 구독 유도 카피 3안 작성 → 팀장 검토 후 1안 라이브
3. **부족 세계관 브리프 v1 + 큐레이션 3건** (tribe-intel)
   - PLAN.md 부족 정의를 운영 기준으로 구체화
   - 지난 7일 인기 #행궁동 포스트 3건 선별 → content에 전달
4. **sitemap.ts에 /privacy 포함 여부 확인 + Search Console 등록** (seo-deploy)
   - sitemap 누락 URL 보완
   - Search Console 등록 → sitemap 제출 (Week 2 첫 항목으로 가던 것을 앞당김)

**P1 (가능하면 6/11까지)**
5. **AdSense 신청 준비** (seo-deploy)
   - 인프라가 이미 갖춰져 있으므로 콘텐츠 2~3편 발행 직후 즉시 신청 가능
   - Week 2 첫날(6/12)에 신청 제출이 목표. 단, 6/11에 콘텐츠 3편 라이브면 6/11에 신청해도 됨.
6. **GA 초기 설정 + 5개 지표 베이스라인** (analytics)
   - GA에 카카오 채널 구독 이벤트 정의
   - 6/11 기준 베이스라인 스냅샷 저장

**P2 (Week 1 안에 못해도 OK, Week 2로 이월)**
- 콘텐츠 4편 이상 발행
- OG 이미지 디자인 개선

### 팀장 발행 승인 프로토콜

content는 5편 초안 → 팀장 리마커블 검토 → 통과한 것만 라이브. **수량 집착 금지.** 6/11까지 2편만 통과해도 OK.

## 의제 3: 핵심 리스크 크로스체크

| 리스크 | 발생 시 영향 | 대응 |
|--------|------------|------|
| **카카오 채널 알림 비용** | 무료 알림 한도 초과 시 메시지 비용 발생 | Month 1에는 친구톡 발송 최소화. 주 1회 뉴스레터만. 50명까지는 무료 한도 내. |
| **AdSense 거절** | 수익화 지연 + 사이트 품질 문제 시그널 | 거절 사유 분석 → 72시간 내 개선안. 지금 인프라 상태면 거절 가능성 낮음. |
| **콘텐츠 리마커블 실패** | 3편 연속 공유 0 → 자기수정 트리거 | 팀장이 매 글 검토. 카톡방 공유 가능성 없으면 반려. 수량보다 통과 여부. |
| **부족 규범 자기충족 실패** | "남들보다 잘 안다" 정체성이 만들어지지 않음 | tribe-intel 주간 브리프로 현실 검증. Month 1 말 재방문율 20% 미달 시 부족 정의 재탐사. |
| **AI 톤 진부화** | 한국 단골 톤이 너무 매끈하면 진정성 의심 | content는 "구어체·시간정보·특정 가게 이름"을 매 글 1개 이상 포함. 익명성·일반화 회피. |
| **네이버 인덱싱 지연** | Google만 잡고 한국 사용자 미도달 | Week 2부터 네이버 블로그 교차 발행. Week 1에는 우선순위 아님. |
| **글로벌 트래픽 무시의 기회비용** | Month 3 회고에서 후회 가능성 | seo-deploy가 Month 3 회고에 "글로벌 무관심 유입" 통계 별도 보고. 10%/월 넘으면 i18n 검토. |

### 팀장 추가 경고 (Seth Godin 금기 항목)

- "수량 집착" — 콘텐츠 주 5편 시도 → 2~3편 발행. 5편 강행 금지.
- "퍼널 없이 광고 집행" — 카카오 채널이 라이브되기 전 어떤 유료 광고도 집행 금지.
- "즉각 ROI" — Month 1 광고 수익 0원이어도 정상. 부족 깊이가 일차 지표.

1. **부족 = 수원·경기 거주 행궁동 단골 한국인**. 한국어 단일 타겟. hreflang/번역 작업 일체 금지.
