## 1. 쓰레드 (메인)

### 본문

```
행궁동 주차장 만차 보고 20분 빙빙 돌다가
답답해서 사이트 하나 만들어버림

행궁동 공영주차장
실시간 잔여석 / 요금 / 내 위치에서 가까운 순
한 화면에 다 나옴

👉 [링크]

행궁동 주차 때문에 고생한 사람 나만 아니지?

#행궁동 #수원여행 #행궁동주차
```

### 셀프 댓글 1 (올리자마자 바로)

```
주차장이란 주차장은 다 만차,
잠깐 세워두고 폰으로 검색할 데도 없고
골목에서 갇혀서 진짜 현타 왔음
집 와서 바로 만들기 시작함
```

### 셀프 댓글 2 (1분 후)

```
무료이고 광고 없음
주차장 앞에서 멘탈 나가는 건 나 하나로 충분하니까
주말에 행궁동 갈 일 있으면 가기 전에 확인해보세요
```

### 게시 후 액션
1. **프로필 고정** — 이 포스트를 프로필 상단에 고정
2. **셀프 댓글 2개** — 초기 알고리즘 부스트
3. **행궁동 관련 계정 댓글 활동** — 맛집/카페 계정 포스트에 자연스러운 댓글 → 프로필 유입 유도
   - 예: "저도 행궁동 자주 가는데 주차 때문에 항상 스트레스예요 ㅋㅋ"
   - 광고 톤 X, 공감 톤 O
